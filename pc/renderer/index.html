BeastSaber Downloader

Send lists over the internet relay, over Wi‑Fi (LAN), or import from file—then download ZIPs to a folder you choose.

ZIP handling

Used for manual downloads on Import from file and for phone auto-download (Internet relay or Receive over LAN when enabled).

Folders match the ZIP name (e.g. key — Song). Enable delete only if you keep the extracted songs.

Internet relay

When the phone is not on the same Wi‑Fi, the relay queues your list until this PC picks it up. You can self-host the server from repo relay/ or use the default URL below.

If this stays off, lists still import—but ZIPs never download automatically over the relay, even if the phone enables auto-download.
